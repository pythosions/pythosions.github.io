## Имитация асинхронного запроса
- При нажатии смотреть полный рейтинг происходит имитация запроса данных с сервера.
- Используется `fetch` для запроса получения данных из "локальных `.json` файлов" - фактически же данные генерируются рандомно специально созданным mock-файлом.
- В `fetch` добавлен искусственный `setTimeout` для имитации сетевой задержки, исходящий из количества элементов в запросе, что делает поведение приложения более реалистичным.
- Контент таблицы динамически перерисовывается на основе полученных данных без перезагрузки страницы.

##  Интерактивные UI-элементы
- **Анимация бейджев:** при наведении курсора на бейджи  срабатывает плавная CSS-анимация проблеска.
- **Анимация звезд рейтинга:** звезды имеют анимацию при наведении, чтобы показать их интерактивность.
- Все анимации реализованы на чистом CSS с использованием `transition` для плавности и производительности.


##  Кросс-браузерность
- Протестировано и корректно работает в последних версиях браузеров:
  - **Google Chrome**
  - **Mozilla Firefox**
  - **Safari**
